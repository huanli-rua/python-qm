
# 期末作业 - 计算机系教师研究方向文本整理与统计分析系统

## 任务背景
在计算机科学研究中，常需要对大量非结构化文本（如教师研究方向介绍、研究概况、论文摘要等）进行自动整理与分析。本作业要求你使用 Python，对清华大学计算机科学与技术系在职教师的研究领域与研究概况文本进行采集、清洗、关键词识别与统计分析，并以规范的文件形式输出结果。

## 任务要求
- 爬取清华大学计算机科学与技术系教师的研究领域与研究概况文本。
- 清洗文本，去除空行、多余空格等。
- 根据关键词文件对文本进行关键词识别，输出结果到文件。
- 汇总统计每个关键词被多少位教师命中，生成汇总文件。
- 实现命令行查询功能，包括列出关键词及其命中数量和查询教师名单。

## 目录结构
- `data/`：爬取生成的原始文本
- `cleaned/`：清洗后的文本
- `keywords/`：每位教师的关键词识别结果
- `summary.txt`：汇总统计文件
- `keywords.txt`：关键词文件
- `main.py`：程序入口
